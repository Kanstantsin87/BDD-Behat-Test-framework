BDD-Behat-Ticketmaster Test automation framework for ticketmaster.co.uk. Used Behat+Mink.
----------------

Installing additional packages
----------------
```bash
$> composer update
```

Run auto-tests:
----------------
```bash
$> bin/behat
```

To get junit .xml report run
----------------
```bash
$> bin/behat -f junit
```

Test result reports in folder ./build
